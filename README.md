#### For Windows Users:

### Step 5: Start Adding Recipes!

Once logged in, you can start adding your favorite recipes. Click on the "Add Recipe" button to begin. Fill in the fields provided such as recipe name, ingredients, and steps. Save your recipe to see it in your collection!
